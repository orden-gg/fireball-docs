## Setup

1. Create a [Gigaverse](https://gigaverse.io) account — this gives you an Abstract wallet
2. Go to [fireball.gg](https://fireball.gg) and sign in with that same wallet
3. Sign the authorization message — this lets Fireball make in-game moves on your behalf, **not** wallet transactions
4. Head to **Quests** and claim the **Scroll of Noob** for a 5-day free trial
5. Make sure your **Giga-Juice** is active in Gigaverse to unlock full access after the trial
6. After the trial, purchase [Sparks](/guides/sparks) to keep playing

## Your First Run

### Dungeon

1. Go to **Play → Dungeon**
2. Pick a dungeon type — start with **Normal**
3. Choose an AI preset — **Balanced** is a safe default
4. Set the number of runs and hit **Start**

### Fishing

1. Go to **Play → Fishing**
2. Choose a fishing type — **Normal** is a good starting point
3. Select **Agent** for automated play or **Manual** for hint-assisted play
4. Set the number of sessions and hit **Start**

Once running, your jobs appear in the **Jobs** page. You can close the browser — Fireball runs everything server-side.
