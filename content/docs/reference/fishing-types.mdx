A quick-reference table for all fishing types available in Fireball.

## Overview

| Type       | Energy Cost | Daily Limit | Juiced Limit | Description                                   |
| ---------- | ----------- | ----------- | ------------ | --------------------------------------------- |
| **Small**  | 12          | 10          | 20           | Lower difficulty, smaller fish, lower rewards |
| **Normal** | 16          | 10          | 20           | Balanced difficulty and reward pool           |
| **Big**    | 20          | 10          | 20           | Toughest fish, highest rewards                |

## Juiced Mode

All fishing types support juiced mode, which doubles the daily limit from 10 to 20 sessions.

## Gameplay Modes

Both **Agent** (automated) and **Manual** (player-controlled with hints) modes are available for all fishing types.

| Mode       | Sessions Per Start  | Agent AI Control                    | Hints                            |
| ---------- | ------------------- | ----------------------------- | -------------------------------- |
| **Agent**  | 1–20 (configurable) | Full automation via Fireball agent AI | N/A                              |
| **Manual** | 1                   | Player controls all decisions | Available with confidence levels |

## Equipment

Each fishing session uses:

- **1 Rod** — Primary fishing weapon
- **Up to 2 Lures** — Secondary fishing equipment

Equipment defaults to your currently equipped gear but can be customized per session.
