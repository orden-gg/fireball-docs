A quick-reference table for all dungeon types available in Fireball.

## Overview

| Type          | Rooms | Energy Cost | Daily Limit | Juiced Limit | Juiced Energy | Notes                                 |
| ------------- | ----- | ----------- | ----------- | ------------ | ------------- | ------------------------------------- |
| **Normal**    | 16    | 40          | 10          | 12           | 3x (120)      | Standard dungeon, good starting point |
| **Underhaul** | 16    | 40          | 8           | 9            | 3x (120)      | Requires Underhaul checkpoint unlock  |

## Juiced Mode

Juiced runs are available for both **Normal** and **Underhaul**. Juiced mode costs 3x the normal energy but grants additional daily runs beyond the standard limit.

## Unlock Requirements

| Type      | Requirement                 |
| --------- | --------------------------- |
| Normal    | None                        |
| Underhaul | Underhaul checkpoint unlock |

See [Games](/gigaverse/games) for an overview of dungeon types.
