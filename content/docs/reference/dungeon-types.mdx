A quick-reference table for all dungeon types available in Fireball.

## Overview

| Type          | Rooms | Energy | Daily Limit | Juiced Limit | Notes                                |
| ------------- | ----- | ------ | ----------- | ------------ | ------------------------------------ |
| **Normal**    | 16    | 40     | 10          | 12           |                                      |
| **Gigus**     | 16    | 200    | 30          | 30           | No gear                              |
| **Underhaul** | 16    | 40     | 8           | 9            | Requires Underhaul checkpoint unlock |
| **Void**      | 9     | 20     | 4           | —            | No gear, no juiced mode              |

## Juiced Mode

Juiced mode costs 3x the normal energy and grants additional daily runs. Available for Normal and Underhaul only. Gigus has the same limit juiced or not. Void does not support juiced mode.

## Unlock Requirements

| Type      | Requirement                 |
| --------- | --------------------------- |
| Normal    | None                        |
| Gigus     | None                        |
| Underhaul | Underhaul checkpoint unlock |
| Void      | None                        |

See [Games](/gigaverse/games) for an overview of dungeon types.
