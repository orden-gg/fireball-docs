Fireball is available as a **Progressive Web App (PWA)** — install it from your browser for a native app-like experience with push notifications and quick access.

## Why Install?

- **Quick access** — Launch Fireball from your home screen or app drawer
- **Push notifications** — Receive updates about job completions and events
- **Native feel** — Full-screen experience without browser chrome
- **Earn Sparks** — Complete the PWA Install quest for bonus Spark rewards

## How to Install

### Desktop (Chrome, Edge)

1. Visit [fireball.gg](https://fireball.gg) in your browser
2. Look for the **install icon** in the address bar (usually a monitor with a down arrow)
3. Click it and confirm the installation
4. Fireball will appear as an app in your system

### Mobile (iOS Safari)

1. Open [fireball.gg](https://fireball.gg) in Safari
2. Tap the **Share** button (square with an up arrow)
3. Scroll down and tap **Add to Home Screen**
4. Tap **Add** to confirm

### Mobile (Android Chrome)

1. Open [fireball.gg](https://fireball.gg) in Chrome
2. Tap the **three-dot menu** in the top right
3. Tap **Add to Home Screen** or **Install App**
4. Confirm the installation

## Install Prompt

Fireball may show a custom install prompt in the bottom-right corner of the screen. This appears a few seconds after loading if you haven't installed the app yet.

- Click **Install Now** to proceed
- Click **Maybe Later** to dismiss for 1 day

## PWA Install Quest

After installing the mobile app, head to the [Quests](/guides/quests-and-rewards) page to claim the **PWA Install quest** for bonus Sparks. This quest can only be claimed from the installed mobile app.

## Auto-Updates

The app checks for updates periodically. When a new version is available, you'll be prompted to refresh. Updates apply automatically — no need to reinstall.
