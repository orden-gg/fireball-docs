The biggest Fireball release yet. v1.0.0 ships the installable PWA, a full mobile overhaul, manual fishing with AI hints, and the official documentation site.

### Highlights

- 🔥 **PWA** — Fireball is now installable as a native app on iOS and Android. Push notifications, offline support, and a PWA install quest with spark rewards ([Installing the App](/guides/pwa-install))
- 🔥 **Manual fishing** — Play the card game yourself. AI agent hints with confidence levels, autoclaim, autorepair, and action token support for faster play ([Jobs](/features/jobs))
- 🔥 **Mobile redesign** — Every page in the app has been rebuilt from the ground up for mobile. Navigation, home, fishing, dungeons, world, user profiles, pets — the full application is now mobile-first
- 🔥 **Discord login** — Sign in with your Discord account
- 🔥 **Documentation site** — This site. Guides, feature references, and changelogs now live at fireball.gg/docs

### Also in this release

- Drops feed — live feed of drops across all games with game type and item filters
- Sparks redesign — replaced power with sparks burned/balance across user and leaderboard pages; Scroll of Fire bypasses spark cost ([Sparks](/guides/sparks))
- User profiles — fully redesigned with activity heatmap, stats charts by category, time range selector, profile header with badges, and shop integration ([User Profile](/features/user-profile))
- Pets — feed all with one tap, favorite food, hunger countdown, accurate production calculations ([Pets](/features/pets))
- Settings persist — last dungeon and fishing settings saved to account
