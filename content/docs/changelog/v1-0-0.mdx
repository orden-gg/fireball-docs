### Features

- Add manual fishing mode — play the card game yourself with optional agent AI hints ([Jobs](/features/jobs))
- Redesigned play page with improved layout and configuration flow ([Jobs](/features/jobs))
- Redesigned fishing viewer with mobile-first UI, share cards, and history tracking ([Jobs](/features/jobs))
- New fishing agent AI — improved automated fishing strategy ([Jobs](/features/jobs))
- Manual fishing hints — agent AI-powered card recommendations with confidence levels ([Jobs](/features/jobs))
- Autoclaim and autorepair options now available in manual fishing mode ([Jobs](/features/jobs))
- Redesigned user profiles with stats charts, time range selector, and category breakdowns ([User Profile](/features/user-profile))
- Conquest crown leaderboard and user faction highlighting ([World & Crafting](/features/world-and-crafting))
- Conquest UI updates with concluded screen and improved settings ([World & Crafting](/features/world-and-crafting))
- Mobile navigation sidebar and home page redesign ([Dashboard](/features/dashboard))
- Login via Discord
- Free access to games with Scroll of Fire (bypasses spark cost) ([Sparks](/guides/sparks))
- PWA install quest with spark rewards ([Installing the App](/guides/pwa-install))
- Pet feed popover replaced with in-card overlay ([Pets](/features/pets))
- Action token support for faster manual gameplay ([Jobs](/features/jobs))
