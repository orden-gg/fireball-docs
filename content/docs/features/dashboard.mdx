The Fireball dashboard is your command center. After signing in, this is where you access every feature — from starting game runs to managing pets, crafting items, and tracking your stats.

## Sidebar Navigation

The collapsible sidebar gives you quick access to every section:

### Main

| Section       | Description                                                                                             |
| ------------- | ------------------------------------------------------------------------------------------------------- |
| **Play**      | Launch dungeon, fishing, or Conquest sessions — see [Games](/gigaverse/games)                           |
| **Jobs**      | Monitor your [active and past automation jobs](/features/jobs)                                          |
| **Inventory** | Browse your [items](/features/inventory), filter by type, search, and see total value                   |
| **Pets**      | Manage [pets and eggs](/features/pets-and-eggs) — feed, hatch, and filter                               |
| **World**     | Access [crafting](/features/world-and-crafting): Alchemy, Workbench, Merchant, Cooking, Hatchery, Stubs |
| **Services**  | Set up [automated services](/features/services): Fish Seller, ROM Claimer, Loot Collector, Hatchery       |

### Games

| Section      | Description                                              |
| ------------ | -------------------------------------------------------- |
| **Dungeons** | View dungeon run history with type filters and win rates |
| **Fishing**  | View fishing history with type filters and catch rates   |

### More

| Section       | Description                                                       |
| ------------- | ----------------------------------------------------------------- |
| **ROMs**      | Manage your [ROMs](/features/roms) and view ROM Core energy/stats |
| **Quests**    | Complete [quests](/guides/quests-and-rewards) and claim rewards   |
| **Referrals** | Get your [referral link](/guides/referrals) and track earnings    |

### Bottom Navigation

| Section     | Description                             |
| ----------- | --------------------------------------- |
| **Home**    | Public landing page with platform stats |
| **Leaders** | Leaderboard ranked by XP                |
| **FAQ**     | Frequently asked questions              |

## Active Job Indicator

When a job is running (dungeon or fishing), the sidebar shows an **animated border** around the Jobs section — a quick visual cue that something is in progress.

## Account Switcher

If you have multiple Gigaverse accounts under one wallet, the **account switcher** in the sidebar lets you swap between them. Your selected account determines which data, jobs, and settings you see.

## Giga-Juice & Session Status

The sidebar displays your current **Giga-Juice status** with an expiration countdown. If your session is limited or expired, a banner appears at the top of the dashboard.

## Daily Reset Countdown

A countdown timer in the navigation shows when the daily reset occurs — this is when your dungeon and fishing daily limits refresh.

## Mobile Navigation

On mobile, the sidebar collapses into a **dock navigation** at the bottom of the screen with quick access to the most-used sections. Additional pages are accessible through the menu.
