---
title: User Profile
description: View player stats, game breakdowns, activity heatmaps, and collections
---

Every Fireball player has a public profile page showing their stats, game history, and collections. Visit any player's profile at `fireball.gg/user/{name}`.

## Stats Grid

The profile header displays four key metrics:

| Stat            | Description                                                          |
| --------------- | -------------------------------------------------------------------- |
| **Bag Value**   | Total inventory value based on last sale prices                      |
| **Total Games** | All games played across every mode                                   |
| **ROMs**        | ROM count with a tier distribution bar (Silver / Gold / Void / Giga) |
| **Pets**        | Hatched pet count with a rarity distribution bar                     |

## Game Breakdown

Three cards show performance per game mode:

### D5000 (Normal Dungeons)

- Games played, wins, win rate percentage
- Skill level (if available)

### Underhaul

- Games played, wins, win rate percentage
- Skill level (if available)

### Fishing

- Games played, wins, win rate percentage
- Skill level (if available)

## Activity Heatmap

A visual heatmap showing daily activity. Toggle the time range:

| Range   | Shows            |
| ------- | ---------------- |
| **7d**  | Last 7 days      |
| **30d** | Last 30 days     |
| **All** | Complete history |

The heatmap updates based on the selected game type.

## Profile Tabs

Below the stats, profiles have additional tabs:

| Tab        | Content                        |
| ---------- | ------------------------------ |
| **Pets**   | Player's pet collection        |
| **ROMs**   | Player's ROM holdings          |
| **Skills** | Skill levels across game modes |
| **Bag**    | Inventory items                |
| **Games**  | Game history                   |

## Public Leaderboard

The **Leaders** page (`fireball.gg/leaders`) shows all players ranked by XP. The leaderboard is:

- Paginated and searchable
- Filterable to show only Fireball accounts
- Sortable by XP
