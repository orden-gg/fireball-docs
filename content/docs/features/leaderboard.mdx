The Leaderboard ranks every Gigaverse account on a single public page — no login required. Sort by Sparks burned, inventory value, games played, or referrals to see who's on top.

## Columns

| Column | Description |
| --- | --- |
| **Rank** | Ordinal position based on current sort — XP rank by default |
| **User** | Account name with a Fireball badge if the account is a verified Fireball account |
| **Sparks Burned** | Total XP accumulated (base Epoch_0 XP plus sparks burned through gameplay) |
| **Referrals** | Number of approved referrals the player has made |
| **Games** | Total games played — hover or tap for a dungeon / fishing breakdown |
| **Inventory** | Estimated total inventory value in USD, calculated from last-sale prices |

## Sorting

Click any column header to sort the leaderboard by that stat. Click again to reverse the order. Available sort fields:

- **Sparks Burned** (default — descending)
- **Inventory Value**
- **Games**
- **Dungeon Games**
- **Fishing Games**
- **Referrals**
- **Name**

## Filters

### Search by Name

Type in the search box to filter results by player name. The search is debounced and updates automatically as you type.

### Fireball Accounts Only

Check **Fireball accounts** to hide unverified accounts and see only players using the Fireball platform.

## Pagination

Results are shown 20 per page. Use the pagination controls at the bottom of the table to move between pages.

## Games Breakdown

The **Games** column shows total game count. Hovering (or tapping on mobile) opens a detail card with:

- Dungeon games count
- Fishing games count

## Inventory Value

Inventory value is an estimated USD figure derived from the last recorded sale price of each item in the player's wallet, multiplied by quantity. It reflects on-chain market data and updates periodically — it is not real-time.

## Sparks Burned

The **Sparks Burned** column represents a player's total XP, which combines:

- **Base XP** — awarded from Epoch_0 participation
- **Games XP** — sparks burned through dungeon and fishing runs

This is the primary measure of how active a player has been on the platform.

## Fireball Badge

Accounts verified as Fireball users display a badge next to their name. Use the **Fireball accounts** filter to view only this group.
