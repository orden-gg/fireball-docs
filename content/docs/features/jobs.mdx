---
title: Jobs
description: Monitor and manage your active and past automation jobs
---

Every time you start a dungeon run or fishing session, Fireball creates a **job** — a tracked task that runs on Fireball's servers. The Jobs page is your control center for monitoring progress and reviewing results.

## Job Statuses

| Status          | Color  | Meaning                                       |
| --------------- | ------ | --------------------------------------------- |
| **In Progress** | Green  | Currently running, animated border in sidebar |
| **Queued**      | Yellow | Waiting to start                              |
| **Paused**      | Gray   | Temporarily stopped                           |
| **Completed**   | Green  | All runs finished successfully                |
| **Failed**      | Red    | Encountered an error (hover for details)      |
| **Cancelled**   | Yellow | Manually stopped                              |

When a job is in progress or queued, the **Jobs** item in the sidebar shows an animated border trail — a quick visual indicator that something is running.

## Jobs List

The main Jobs page shows all your recent jobs with:

- **Time** — When the job was created (relative format like "2 hours ago")
- **Status** — Color-coded badge
- **Progress** — Completed runs vs. total (e.g., "5 / 10")
- **Win rate** — Percentage badge for completed runs
- **Game type** — Dungeon or Fishing badge
- **Configuration** — Dungeon type, model, juiced mode, or fishing type and model

### Filters

| Filter        | Options                                                 |
| ------------- | ------------------------------------------------------- |
| **Limit**     | 20, 50, or 100 latest jobs                              |
| **Game type** | All Types, Dungeon, or Fishing                          |
| **Refresh**   | Manual refresh button (auto-refreshes every 30 seconds) |

## Job Detail Page

Click any job to see its detail view. The detail page shows:

- Full job configuration and status
- Individual run results (dungeon-by-dungeon or fishing-by-fishing)
- Links to the live viewer for in-progress runs
- Auto-refreshes every 15 seconds while the job is running (stops when completed)

## Watching Live

From the job detail page, you can open the **live viewer** to watch a dungeon run or fishing session in real time. See [Games](/gigaverse/games) for details on each game mode.
