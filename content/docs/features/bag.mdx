The Inventory page shows everything you own in Gigaverse — materials, consumables, fish, gear, collectables, and more. Browse your items at a glance or drill down with filters and search.

## Item Grid

Your inventory is displayed as a grid of item cards. Each card shows:

- Item icon and name
- Quantity owned
- Rarity indicator

Items with a zero balance are hidden automatically.

## Filtering

### By Type

Filter your inventory by item category:

| Type             | Description                        |
| ---------------- | ---------------------------------- |
| **Materials**    | Crafting materials and resources   |
| **Consumables**  | Usable items like potions and food |
| **Skins**        | Cosmetic items                     |
| **Rings**        | Ring equipment                     |
| **Fish**         | Caught fish by species             |
| **Pet Produce**  | Items produced by pets             |
| **Pet Food**     | Food items for feeding pets        |
| **Collectables** | Collectable items                  |
| **Redeemables**  | Items that can be redeemed         |
| **Other**        | Everything else                    |

On desktop, type filters appear as a toggle group. On mobile, they're in a popover menu.

### By Name

Use the **search bar** to filter items by name. The search is case-insensitive and updates in real time as you type.

### Reset

A **reset button** appears when any filters are active, clearing all filters back to the default view.

## Sorting

Items are sorted by **rarity** (highest first), then by **balance** (lowest first within the same rarity). This puts your rarest items at the top.

## Total Value

The inventory header alternates between displaying:

- **Total inventory value** — Calculated from the last sale price of each item multiplied by your quantity
- **Total item count** — How many distinct items you own
